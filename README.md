## 快速开始

### 从源代码开始

```commandline
git clone https://github.com/pykelysia/python-chat.git
cd python-chat
pip install zai
python python.py
```

### 使用可执行文件

选择 `Release` 中的任意版本，下载相应可执行文件，并解压运行。
